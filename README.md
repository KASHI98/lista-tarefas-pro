# 🚀 Lista de Tarefas Pro

Bem-vindo ao **Lista de Tarefas Pro** — um sistema simples, rápido e moderno para organizar suas tarefas diárias diretamente no navegador.

🌐 **Use agora:**
👉 https://kashi98.github.io/lista-tarefas-pro/

---

## ✨ Sobre o projeto

O **Lista de Tarefas Pro** foi criado para ajudar pessoas a organizarem suas atividades de forma prática, visual e rápida.

✔️ Sem login
✔️ Sem instalação
✔️ Funciona em celular e computador
✔️ Dados salvos automaticamente

---

## 🧠 Funcionalidades

* ✅ Criar tarefas
* ⭐ Favoritar tarefas importantes
* 📌 Fixar tarefas no topo
* 🎨 Escolher cor personalizada
* ✏️ Editar tarefas
* 🗑️ Excluir tarefas
* 💾 Salvamento automático (LocalStorage)

---

## 📱 Responsivo

O sistema funciona perfeitamente em:

* 💻 Desktop
* 📱 Celulares
* 📟 Tablets
